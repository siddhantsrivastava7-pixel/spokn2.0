# Spokn

Offline, privacy-focused speech-to-text for your desktop.

Press a shortcut, speak, and your words appear in any text field — all processed locally on your machine.

## Status

Early fork in active development. Forked from [cjpais/Handy](https://github.com/cjpais/Handy) (MIT).

## Development

```bash
bun install
bun run tauri dev
```
